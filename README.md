# 📊 Student Performance Analysis

Data analysis project using Python, Pandas, Matplotlib, and Seaborn

## 📌 Project Overview

This project analyzes student performance using Python. It calculates total marks, average scores, grades, and generates meaningful insights through data visualization.


## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 🔥 Features

* 📊 Data analysis of student marks
* 🏆 Subject-wise topper identification
* 📈 Class performance insights
* 🥇 Overall top student detection
* 📉 Student ranking system
* ⚠️ Weak student identification
* 🎯 Performance categorization
* 📊 Visualizations (bar chart, heatmap)

## 📂 Dataset

The dataset contains marks of students in the following subjects:

* Maths
* Science
* English
* Computer
* German

*Sample dataset included for demonstration purposes.*

## 🚀 How to Run

1. Install required libraries:

pip install pandas matplotlib seaborn

2. Run the script:

python analysis.py

## 💡 Insights

* Identifies the best performing student
* Highlights subject-wise toppers
* Shows overall class performance trends
